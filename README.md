# 🎮 坦克大战 (Tank Battle)

一个纯 HTML/CSS/JavaScript 实现的经典坦克大战（Battle City）网页游戏。

## 🕹️ 游戏操作

| 按键 | 功能 |
|------|------|
| `↑` `↓` `←` `→` | 移动坦克 |
| `W` `A` `S` `D` | 移动坦克（备选） |
| `空格` | 发射子弹 |
| `R` | 重新开始游戏 |

📱 同时支持触摸屏操作！

## 🎯 游戏玩法

- 消灭关卡中所有敌方坦克即可过关
- 共有 **5 个关卡**，难度递增
- 保护你的基地（地图底部的鹰标志），基地被摧毁则游戏失败
- 每消灭一个敌人获得 **100 分**
- 有 **3 条命**，被击中会重生

## 🗺️ 地图元素

| 元素 | 说明 |
|------|------|
| 🧱 砖墙 | 可被子弹摧毁 |
| 🔩 钢铁墙 | 坚固，无法摧毁 |
| 🌊 水域 | 阻止坦克通行 |
| 🌲 树林 | 遮挡视线 |
| 🦅 基地 | 保护目标！ |

## 🚀 快速开始

直接在浏览器中打开 `tank-battle.html` 即可开始游戏！

```bash
# 或者使用任意 HTTP 服务器
python -m http.server 8000
# 然后访问 http://localhost:8000/tank-battle.html
```

## 🎨 特色

- 经典 Battle City 风格画面

- 3 种不同类型的敌方坦克
